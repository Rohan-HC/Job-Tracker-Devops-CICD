# Job Tracker DevOps CI/CD

A Spring Boot Job Application Tracker deployed through a practical CI/CD pipeline.

## Goal

Build, test, containerise, and deploy a Java application automatically whenever code is pushed to GitHub.

## Planned Pipeline

GitHub -> Jenkins -> Maven build and tests -> Docker image -> Running container

## Technology

- Java and Spring Boot
- Maven
- Docker
- Jenkins
- GitHub
- GitHub Actions
- AWS EC2 later
